Babilejo

Chat across multiple languages

Babilejo is a chat application that leverages the power of the Google Translate API and socket.io to connect people. Users are able to select their native language and receive chats translated into their native language, regardless of the language the message was written in.

Technologies used: Node.js, MongoDB, socket.io, Google Translate API
