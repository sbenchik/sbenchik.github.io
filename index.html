Hi! I'm Steve Benchik, an amateur software developer from South Bend, IN with a passion for problem solving and innovation.

Here's a little bit more about me.

I'm currently a freshman studying computer science at Purdue University. My experience with programming started when I was 13 years old and I produced a my own Ubuntu respin (which still got 2 downloads this year on SourceForge!). Since then I've been interested in the Linux desktop, and my goal is to get involved with open source projects like GNOME, Cinnamon, and KDE. I wrote Codex, a text editor for programmers, using Python 3 and Qt 5. I'm using this website (and probably some other projects) to teach myself HTML, CSS, Javascript, and jQuery.

Codex, a text editor for programmers
